用CSS3实现瀑布流布局

发表于 2019-04-03 |
字数统计: 453 | 阅读时长 ≈ 3

以前使用瀑布流都要用js，现在有了css3，可以轻松实现了。

掌握点：

1、column-count 把div中的文本分为多少列

2、column-width 规定列宽

3、column-gap 规定列间隙

4、break-inside: avoid; 避免元素内部断行并产生新列

注意: Internet Explorer 9及更早 IE 版本浏览器不支持 column-count 属性。
